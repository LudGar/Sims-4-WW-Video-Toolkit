📋
Full Workflow — End to End

    

      

        
1
Download the EA VP6 Encoder Toolkit
Get it from github.com/bluesky-dev12 — Release 4.0. Extract so your folder contains tools\, MP4\ and the .bat at the same level.

        
2
Drop MP4 files into MP4\
No spaces in filenames! Rename My Video.mp4 → MyVideo.mp4. Multiple files are processed in one run.

        
3
Configure & Download the .bat (Studio → section 5–6)
Set resolution, fps and bitrate in the Studio tab. The .bat preview updates live. Download it and place it in the encoder root folder.

        
4
Run VP6_SIMS4_WW.bat
Double-click. Runs 5 steps: FFmpeg downscale → WAV → EA audio stream → nihav-encoder VP6 encode → QuickBMS mux. Output .avi files appear in Release\. Output .avi files appear in Release\.

        
5
Import .avi into S4PE
Open your .package in S4PE. Import each .avi from Release\ — type 376840D7, group 00000000. Copy the hex Instance ID S4PE assigns.

        
6
Fill in Studio tab & Generate XML
Paste the Instance ID into section 4, fill package name, title, author, orientation and duration in sections 1–3, then click Generate XML.

        
7
Import the XML into your .package
In S4PE import the downloaded .xml — type 5B02819E, group 00000000, instance = FNV-64 hex shown next to the package name field.

        
8
Generate Thumbnail (Studio → section 7)
The thumbnail auto-fills from your title/author/tags. Pick a style, optionally upload a background image, download the PNG to use as a preview or share with your mod.

        
9
Save .package & Test In-Game
Drop the .package into your Mods\ folder. Launch Sims 4, have a Sim watch WickedWhims porn TV — your video should appear in the selector!

      

      
Tip: The .bat processes every .mp4 in MP4\ in one run — batch encode multiple videos, then generate a separate XML for each one in the Studio tab.
