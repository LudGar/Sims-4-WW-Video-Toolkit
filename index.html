Full Workflow — End to End

    

      

        
1
Configure Studio tab
Fill in all fields: video filename, title, author, orientations, duration, encoding settings. Everything bakes into the .bat on the fly.

        
2
Drop files into folders
MP4\YourVideo.mp4 — the source file. Thumbnail\YourThumb.png — optional, for DDS conversion. Both folders are auto-created on first run.

        
3
Download & run VP6_SIMS4_WW.bat
One click runs everything: Y4M encode → WAV → EA audio → VP6 → mux → rename .avi. Thumbnail PNG → DDS. XML files written. All outputs go to Release\.

        
4
Import .avi into S4PE
Open your .package in S4PE. Import Release\VideoName.avi — type 376840D7, group 00000000. Copy the hex Instance ID.

        
5
Paste Instance ID → re-download bat
Paste the ID into Step 2 of the Studio tab. Re-download the .bat — now the XML files it writes will reference the correct video resource.

        
6
Re-run the bat (skip encode)
The bat detects Release\VideoName.avi already exists and skips the encode. It just writes the XML files and converts the thumbnail.

        
7
Import XMLs & DDS into S4PE
Import the 3 XML files from Release\ with their correct type IDs. Import the .DDS as type 00B2D882, group 00000000, instance = same as the video.

        
8
Save .package & test in-game
Drop the .package into Mods\. Launch Sims 4 — your video should appear in the WickedWhims porn TV channel.

      

      
Tip: Re-running the bat after adding the Instance ID is fast — encode is skipped, only XML and DDS steps run (~seconds).
